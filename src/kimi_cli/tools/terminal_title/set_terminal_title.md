Set the terminal tab title to a short task name.

Call this tool when the user assigns a new task, or when you start a
significant new phase that deserves its own title. Keep the title to
5 words or fewer — it must fit in a terminal tab.

Examples:
- "Refactor auth module"
- "Fix login bug"
- "Add test coverage"
- "Investigate build failure"

Do not call this tool repeatedly for the same task. One call per task
is enough.
